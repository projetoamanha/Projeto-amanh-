# Projeto O Amanhã

Olá! Este é o repositório do site do **Projeto O Amanhã**, uma ONG focada em revitalização urbana e apoio comunitário em Conservatória, Valença - RJ.

## Sobre o projeto
A ONG faz um trabalho sensacional pintando fachadas de casas de famílias de baixa renda de forma totalmente gratuita, além de promover ações sociais e distribuir doações na comunidade.

Eu desenvolvi esse site como um projeto de extensão do meu curso de Sistemas de Informação. O objetivo principal foi criar uma página moderna e direta ao ponto para ajudar o projeto a ganhar mais visibilidade, captar recursos (doações de tintas ou PIX) e facilitar a entrada de novos voluntários.

## Tecnologias utilizadas
O site foi construído para ser leve, rápido e não depender de servidores complexos. Para isso, usei apenas:
* HTML5 estruturado
* CSS3 puro (com variáveis, flexbox e grid)
* JavaScript puro (para o menu no celular, animações na rolagem da página e um script que envia mensagens de contato direto para o WhatsApp)

## Como rodar localmente
Não é preciso instalar absolutamente nada. Basta fazer o download dos arquivos deste repositório e abrir o arquivo `index.html` em qualquer navegador (Chrome, Edge, Safari, etc).

## Onde o site está hospedado
Como a ideia é não gerar custos para a ONG, o site foi estruturado para ser um site estático e está hospedado gratuitamente usando o GitHub Pages.
